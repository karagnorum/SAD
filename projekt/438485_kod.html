Zadanie 3. a) ElasticNet to model regresji liniowej, który łączy właściwości regresji grzbietowej (ridge regression) i regresji lasso (Least Absolute Shrinkage and Selection Operator). Jego głównym celem jest jednoczesna selekcja cech oraz regularyzacja modelu, co pozwala na uzyskanie bardziej stabilnych i interpretowalnych wyników w przypadku danych o wysokiej kolinearności.

Funkcja, którą optymalizuje model ElasticNet, jest zdefiniowana jako:

\[ \min_{\beta_0, \beta} \left\{ \frac{1}{2N} \sum_{i=1}^{N} (yi - \beta_0 - \mathbf{x}_i^T \beta)^2 + \lambda \left( \alpha \|\beta\|_1 + \frac{1-\alpha}{2} \|\beta\|_2^2 \right) \right\} \]

gdzie: - \(yi\) to wartości zmiennej objaśnianej, - \(\mathbf{x}_i\) to wektor cech dla obserwacji \(i\),

Model ElasticNet estymuje następujące parametry: - \(\beta_0\) - wyraz wolny modelu, - \(\beta_1, \beta_2, \ldots, \beta_p\) - współczynniki regresji dla każdej z \(p\) cech.

ElasticNet posiada dwa hiperparametry: - \(\lambda\) - parametr regularyzacji, który decyduje o łącznej sile regularyzacji, - \(\alpha\) - parametr mieszania, który kontroluje równowagę między regularyzacją L1 i L2.

Dla różnych wartości hiperparametrów model ElasticNet przyjmuje postać regresji grzbietowej lub lasso: - Gdy \(\alpha = 0\), model staje się regresją grzbietową (ridge regression), a - Gdy \(\alpha = 1\), model staje się regresją lasso

  1. Użyjemy 10-fold, aby uzyskać balans między minimalizowaniem wariancji i zwiększaniem elastyczności.
library(glmnet)
x = data.matrix(x_train)
nfolds = 10
foldid = sample(cut(1:nrow(x), nfolds, labels = F))
alphas = seq(from = 0, to = 1, by = 0.5)
lambdas = seq(from = 0.5, to = 1.5, by = 0.5)
new_array = function() array(dim = c(length(alphas), length(lambdas), nfolds))
validation_errors = new_array()
train_errors = numeric(nfolds)
min_error = Inf
for (i in 1:length(alphas)){
  for (j in 1:length(lambdas)){
    test_errors = numeric(nfolds)
    for (k in 1:nfolds){
      train = which(foldid != k)
      model = glmnet(x[train,], y[train], lambda = lambdas[j], alpha = alphas[i], family = "gaussian")
      predicted = predict(model, newx = x, type = "response")
      validation_errors[i, j, k] = mean((predicted[-train]-y[-train])^2)
      train_errors[k] = mean((predicted[train]-y[train])^2)
    }
    validation_error = mean(validation_errors[i,j,])
    if (validation_error < min_error){
        which_alpha_best = i
        which_lambda_best= j
        min_error = validation_error
        train_error = mean(train_errors) #train error in best model
      }
  }
}
